## 📌 Basic Troubleshooting Framework

1. **Identify the problem clearly**
2. **Gather information**
   - Device (Windows / Mac / iPhone / Android)
   - Browser (Chrome / Safari / Firefox / Edge)
   - OS version
   - Steps the user already tried
3. **Start with the simplest solutions**
4. **Escalate to more complex solutions if needed**
5. **Document the final solution**

---

> "Thank you for reporting this issue. Let's troubleshoot this together.

2. Which browser are you using? *(Chrome / Safari / Firefox / Edge)*

# 🔧 Common Technical Issues & Solutions

---

## 🌐 Issue 1: Website Not Loading

**Response Script**

Let's try these steps one by one:

### Step 1: Check Internet Connection
- Try opening **google.com**
- If Google doesn't load → it's likely an internet issue
- Restart your **router/modem**

### Step 2: Clear Browser Cache

**Windows**
- Press `Ctrl + Shift + Delete`
- Select **All Time**
- Check:
  - Cookies
  - Cached Images
- Click **Clear Data**

**Mac**
- Press `Command + Shift + Delete`
- Follow the same steps

### Step 3: Try Incognito Mode
- `Ctrl + Shift + N` *(Windows)*
- `Cmd + Shift + N` *(Mac)*
- Try opening the website again

If it works → issue is with **browser cache/cookies**

### Step 4: Try a Different Browser
- Try **Chrome / Firefox / Edge**
- Check if the site loads there

**Follow-up:**  
> "Let me know which step resolves the issue."

---

## 📂 Issue 2: Can't Upload File

**Response Script**

Let's check a few common causes.

### File Size
- Right-click the file → **Properties**
- Check the file size
- Our limit: **[X MB / GB]**

If the file is too large:
- Compress the file using a **compression tool**

### File Type
Check the file extension:

Examples:
- `.jpg`
- `.pdf`
- `.docx`

Accepted formats: **[list formats here]**

### Browser Check
- Try uploading using another browser
- Ensure the browser is **updated**

### Antivirus Interference
- Antivirus may block uploads
- Temporarily disable it
- Try uploading again

**Follow-up:**  
> "Please try these steps and let me know the result."

## 📱 Issue 3: App Keeps Crashing

**Response Script**

> "App crashes can definitely be frustrating. Let's try a few quick fixes."

---

### 📲 Mobile (iPhone / Android)

#### 1. Force Close the App
- **iPhone:** Swipe up from the bottom → close the app
- **Android:** Tap **Recent Apps** → swipe the app away

Reopen the app afterward.

#### 2. Clear App Cache

- Settings → Apps → [App Name] → Storage → Clear Cache

#### 3. Update the App
- Open **App Store / Play Store**
- Search for **[App Name]**
- Tap **Update** if available

#### 4. Reinstall the App
- Delete the app
- Install again from the store
- Log back into your account

---

### 💻 Desktop

1. Restart the application  
2. Check for updates  
3. Reinstall the application  

---

**Follow-up Question**

> "Which device are you using? I'll provide the exact steps for your device."

# 🚨 When to Escalate to Technical Team

Escalate the issue if:

- All standard troubleshooting steps were attempted (**30+ minutes**)
- Issue requires **backend access**
- Requires **database changes**
- There is a **software bug**
- It's a **security concern**
- Multiple users are affected
